## 🚀 Featured projects

### 🏙 [Urban Mobility Simulation — Havana](https://github.com/anakarla99/Simulation_IA_Project)
> Multi-agent simulation of public transport in Havana, estimating population distribution across municipalities over 24 hours.

- Built a **vehicle + company agent system** with route assignment, maintenance logic and budget tracking
- Applied **Simulated Annealing** for optimal stop assignment and **Ant Colony Optimization (ACO)** for route planning
- Used **AI Planning (uniform-cost search)** to automate company decision-making
- Integrated real geospatial data via **OpenStreetMap + Pyrosm** and demographic data from ONEI Cuba

`Python` `Multi-agent` `ACO` `Simulated Annealing` `AI Planning` `OpenStreetMap`
